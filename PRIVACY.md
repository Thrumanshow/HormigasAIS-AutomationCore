# Privacy Policy – HormigasAIS-AutomationCore

_Last updated: 2025_

## 1. Introduction
HormigasAIS-AutomationCore (“the App”) is a GitHub App developed and maintained by **Thrumanshow**.

This Privacy Policy explains how data is accessed, used, and protected when the App is installed or used.

## 2. Data Access
The App only accesses data explicitly granted through GitHub App permissions during installation. This may include:
- Repository metadata
- GitHub webhook event payloads
- Configuration data required for automation workflows

The App does not access private user information unless explicitly authorized by GitHub permissions.

## 3. Data Usage
Accessed data is used solely to enable automation, workflows, and AI-assisted processes within GitHub.

No data is:
- Sold
- Shared with third parties
- Used for advertising or tracking

## 4. Data Storage
The App does not permanently store personal data unless strictly required for operational automation.

Any temporary data processing is limited to execution of workflows initiated by the user or organization.

## 5. Security
Reasonable technical measures are used to protect data, including:
- HTTPS-secured endpoints
- Webhook secrets
- GitHub-issued authentication tokens

## 6. User Control
Users and organizations may revoke access at any time by uninstalling the App from GitHub.

## 7. Changes
This Privacy Policy may be updated as the App evolves. Updates will be reflected in this document.
